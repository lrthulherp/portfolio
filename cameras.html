Sistema de Monitoramento de Câmeras

Painel estilo NOC com grid dinâmico, fullscreen, tema claro/escuro, login JWT e auto-refresh.

Tecnologias

HTML5 CSS3 JavaScript (ES6+) Bootstrap 5 Node.js Express jsonwebtoken (JWT)

Recursos do Frontend

  • Grid de câmeras com layout responsivo (auto-fill).
  • Fullscreen ao clicar em uma câmera.
  • Tema claro/escuro alternável.
  • Auto-refresh de snapshots (ex.: 15s).
  • Autenticação via JWT (modal de login, verificação e logout).

Recursos do Backend

  • POST /login → autentica e retorna token JWT.
  • GET /verify → valida token.
  • (Opcional) GET /snapshot/:id → proxy seguro para snapshots/streams.

Arquitetura

Navegador (HTML/JS/CSS + Bootstrap) → Node/Express (API JWT) → Câmeras (snapshot/RTSP via proxy opcional).

Para produção, recomenda-se proxy backend para não expor credenciais/URLs das câmeras no front.

Como executar localmente

  1. Instale as dependências e execute o servidor: 
    npm install
node server.js
  2. Acesse http://localhost:3000.
  3. Faça login (ex.: admin/1234 , conforme seu server.js).
